# Mines

## Descrição do Projeto
Uma recriação de um jogo de cassino online, Mines, para uma feira de matemática, nele os usuários podem escolher seu nome de perfil e jogar com um saldo de R$1000,00 fictícios, eles podem escolher a quantidade que deseja apostar e quantas bombas ter no campo, quanto mais bombas colocarem e mais quadrados revelarem, maior será o multiplicador do prêmio.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## Como Executar
Após extrair a pasta:
- Instale a ferramenta XAMPP
- Copie e cole a pasta no caminho (xampp > htdocs > *cole aqui*)
- Abra o painel de controle da ferramenta
- Clique no botão "start" ao lado do texto "MySQL"
- Clique no botão "admin" ao lado do texto "MySQL"
- Clique no botão "Novo" para criar um novo banco de dados
- Dê o nome de "mines" e clique em "Criar"
- Acesse a aba Importar do banco criado
- Faça o upload do arquivo mines.sql e clique em Importar
- Volte para o painel de controle do XAMPP
- Clique no botão "start" ao lado do texto "Apache"
- Abra o seu navegador e acesse o link "http://localhost/mines/".
